Câu lệnh github thường dùng

Câu lệnh Chú thích
git init Tạo ra một Repositories trong máy
git clone Lấy từ trên mạng về
git pull Đồng bộ từ trên mạng về
git add và git add . Thêm vào Repositories trong máy
git commit -m "chú thích" Đồng bộ hóa
git remote add origin https:// Đồng bộ mới
git push và git push -u origin master Tải lên trên mạng
git log check xem file
git status xem trạng thái các file
git log -decorate -graph -oneline ok
